🍞 BreadDesk 웹챗 위젯 테스트

위젯 임베드 방법

아래 스크립트 태그를 웹사이트의 <body> 끝에 추가하세요:

<script src="https://breaddesk.k6s.app/webchat-widget.js" data-server="https://breaddesk.k6s.app"></script>

기능

  • 화면 우하단에 🍞 플로팅 버튼이 표시됩니다
  • 클릭하면 채팅창이 열립니다
  • 이름/이메일 입력 후 채팅을 시작할 수 있습니다 (선택사항)
  • 메시지를 보내면 AI가 자동으로 답변합니다
  • 상담원이 답변하면 30초마다 폴링하여 가져옵니다
  • 세션은 sessionStorage에 저장되어 새로고침해도 유지됩니다

API 엔드포인트

  • POST /api/v1/webchat/sessions - 세션 생성
  • POST /api/v1/webchat/sessions/{sessionId}/messages - 메시지 전송
  • GET /api/v1/webchat/sessions/{sessionId}/messages - 메시지 히스토리

현재 페이지에서 테스트: 화면 우하단의 🍞 버튼을 클릭해보세요!
